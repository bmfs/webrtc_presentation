What is WebRTC?

  • RTC - Real Time Communication
  • Collection of API's & Protocols
  • Enables media capture (audio, video)
  • Supports audio, video and data transfers
  • Peer-to-peer connections
  • ~13 years of development
  • Open Standard
  • Open source implementation (libwebrtc)
    • From Google

Brief Timeline

  • 2011 - First Draft
  • 2013 - First cross browser video call
  • 2014 - Cross browser data transfer
  • 2015 - Safari and Edge support
  • 2020 - 1.0 version

RTCDataChannel

  • Bidirectional peer-to-peer data transfers
  • Similar API to WebSockets
  • Supports unreliable/unordered channels (~UDP)
